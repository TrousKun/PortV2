Statement of Purpose

แรงจูงใจที่ทำให้ผมเลือก

"วิทยาการคอมพิวเตอร์ หรือ Com-Sci "

เนื่องจากว่าผมต้องการเรียนการ Coding จากที่ผมได้ลองและผมรู้สึกชอบวิธีการทำงานของโปรแกรม รู้สึกสนุกไปกับการ Coding

ผมจึงอยากที่จะรู้ถึงการ Coding ขั้นสูงกว่า

การศึกษาด้วยตัวเอง ตามแหล่งที่มาเช่น YT หรือ ตาม Website ต่างๆ

สาเหตุสำคัญของการที่ผมเลือกที่จะฝึกการเขียน Code คือ การที่ตลาดของประเทศและโลก อาชีพอย่าง Programmer และ Software Development เป็นที่ต้องการในตลาดค่อนข้างสูงเนื่องด้วยโลกตอนนี้เทคโนโลยีพัฒนาไปเร็ว ผมจึงอยากที่จะเรียนรู้สิ่งนี้เพื่อนำไปใช้พัฒนาองค์กรที่ผมอาจจะสร้างขึ้นในอนาคต หรือ ไปช่วยองค์กรอื่นพัฒนาการเพื่อตามทันโครงสร้างของโลก ที่เปลี่ยนไปอย่างรวดเร็ว ณ ปัจจุบัน เนื่องจากผมเรียนรู้ด้วยตัวเองมาส่วนนึงผ่าน YT หรือ website อย่าง w3school และ เว็ป Portfolio นี้เป็นการนำ Template ที่มีแจกจาก Website หลักของทาง Bootstrap เพื่อนำมาแก้ไขและลดเวลาในการทำงาน เนื่องจาก ผมสามารถเขียน Basic of HTML หรือ Command พื้นฐานของ Python และกำลังศึกษาเพิ่มเติมเกี่ยวกับ CSS และ JavaScript และผมกำหนด RoadMap ในการ Self-Lerning ของผมไว้แล้วทั้ง FrontEnd และ Backend เช่น Backend เป็น GO PHP C C# C++ Node.JS และ Frontend หลักๆจะเป็น HTML CSS Bootstrap Tailwind JavaScript

สิ่งที่ผมเคยทำที่พอเป็นประสบการณ์

การเขียน Webpage ของบริษัทที่แม่ของผมเคยทำงาน

เรียกว่าเป็นประสบการณ์ครั้งแรกกับการเขียนเว็ปจริงๆจังๆของผมเลยก็ว่าได้ นั่นคือการที่ทางบริษัทที่แม่ของผมกำลังทำงานอยู่ได้มีการเปลี่ยนชื่อบริษัท และจำเป็นจะต้องจ้าง Programmer มาเขียนเว็ปให้ใหม่เพื่อแก้ข้อมูลให้เป็นปัจจุบัน ผมจึงอาสาที่จะขอลองที่จะเขียนเองโดยใช้ Model เว็ปเดิมมาเป็นแบบเนื่องจากเขาต้องการเว็ปแบบเดิมแต่เปลี่ยนข้อมูลภายในเว็ป และผมก็ทำมันสำเร็จถือเป็นประสบการณ์การทำงานของครั้งแรกกับการ Coding ของผมโดยใช้พื้นฐานต่างๆที่ศึกษามาด้วยตัวเองนำมาผสมผสานกันแต่สุดท้ายก็ไม่สามารถใช้งานได้เนื่องจากเป็นเว็ปหลักของทางบริษัทเขาจึงจ้าง Programmer ตัวจริงมาทำ แต่ก็ถือเป็นประสบการณ์นึงของผมที่เคยได้ทำมา
